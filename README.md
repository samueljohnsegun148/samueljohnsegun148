## Current Projects

I am currently a core contributor at the [Shared Reality Lab](https://github.com/Shared-Reality-Lab), where I work on:

- **[IMAGE-server](https://github.com/Shared-Reality-Lab/IMAGE-server):**  
  Python-based server powering accessible, real-time, multi-modal computer interaction. I build robust cloud infrastructure, and research tools for ML-based assistive technology.  
  _[LIVE demo: image.a11y.mcgill.ca](https://image.a11y.mcgill.ca)_

- **[HapticMapExplore](https://github.com/samueljohnsegun148/HapticMapExplore):**  
  Digital map exploration with haptic (force feedback) interfaces in Unity/C#. Innovative nonvisual spatial cues for accessible navigation.

- **[ambient-sound-classification](https://github.com/samueljohnsegun148/ambient-sound-classification):**  
  Robust ML pipeline for environmental audio recognition and scene understanding using Python and neural networks.

- **[IndoorNavigationTACME](https://github.com/samueljohnsegun148/IndoorNavigationTACME) & [IndoorNavigationTACMEAndroidApp](https://github.com/samueljohnsegun148/IndoorNavigationTACMEAndroidApp):**  
  Real-time indoor navigation for visually impaired users, combining smartphone IMU, floor-plan modeling, and an AI conversational agent for dynamic wayfinding. Built with Python, Kotlin, and advanced interface engineering.
